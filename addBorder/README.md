Given a rectangular matrix of characters, add a border of asterisks(\*) to it.

**Example**

For

    picture = ["abc",
           "ded"]

the output should be

    addBorder(picture) = ["*****",
                      "*abc*",
                      "*ded*",
                      "*****"]

**Hints**

- concat()
- unshift()
- push()

**Input/Output**

- **[time limit] 4000ms (js)**
- **[input] array.string picture**
